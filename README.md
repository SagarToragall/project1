# Login & Signup Form (Frontend)

A simple responsive **Login / Register** UI built with **HTML + CSS + JavaScript**.

## Preview / Run locally

### Option A: Open directly
- Open `index.html` in your browser (double-click it).

### Option B: Use a local server (recommended)
- If you use VS Code/Cursor, install the **Live Server** extension.
- Right-click `index.html` → **Open with Live Server**.

## Project structure

- `index.html`: Markup for the login + signup panels.
- `style.css`: Styling, layout, and animations.
- `style.js`: Small JS to switch between Login and Sign Up views.

## Notes

- This project is UI-only; the forms do not submit to a backend yet (`form action="#"`).
